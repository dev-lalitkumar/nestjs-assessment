# Nest.Js Assessment

## Installation

Use the package manager [npm]() to install the dependencies.

```bash
npm install
```

## Environment Variables

```env

JWT_SECRET="A Random String"

DB_HOST="localhost"
DB_PORT=3306
DB_NAME="nestjs-assessment"
DB_USERNAME="root"
DB_PASSWORD="root"
```
